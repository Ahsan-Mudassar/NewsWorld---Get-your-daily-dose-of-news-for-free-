#  NewsWorld - React News App

A modern and responsive news application built with **React** that fetches real-time news using an API. Users can browse the latest headlines by category and stay updated with current events around the world.

---

## Features

*  Latest news headlines
*  Country & category-based news
*  Infinite scrolling (load more on scroll)
*  Fast performance using Vite
*  Clean and responsive UI
*  Optional Dark Mode (if added)

---

## Tech Stack

* **React JS**
* **Vite**
* **JavaScript (ES6+)**
* **CSS / Tailwind CSS (if used)**
* **News API**

---

## Installation & Setup

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/your-username/newsworld.git
```

### 2. Navigate into the project

```bash
cd newsworld
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start development server

```bash
npm run dev
```

---

##  API Setup

This project uses **News API**.

1. Go to https://newsapi.org
2. Create an account and get your API key
3. Replace the API key in your code:

```js
const apiKey = "YOUR_API_KEY";
```
